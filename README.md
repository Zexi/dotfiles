# dotfiles of mine

## use with nvim

```sh
./dothelper install vim
ln -s ~/.vimrc ~/.vim/init.vim
ln -s ~/.vim ~/.config/nvim
```

### complete & goto definitions
```sh
pip install python-language-server
```

```sh
# use mons is recommend
yaourt -S mons
```
